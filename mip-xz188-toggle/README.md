# mip-xz188-toggle

mip-xz188-toggle 用来支持点击按钮显示隐藏切换详情页

标题|内容
----|----
类型|通用
支持布局|N/S
所需脚本|http://mipcache.bdstatic.com/static/mip-xz188-toggle/{版本号}/mip-xz188-toggle.js

## 示例

### 基本用法
```html
<mip-xz188-toggle>
     <i class="hideshowbtn">点我</i>
</mip-xz188-toggle>
```

## 属性
显示隐藏切换位置(hsId)

### {hsId}

说明：{显示隐藏切换位置}
必选项：{是}
